# Reservationsite

<h2>◆概要◆</h2>
予約サイトを構築しました。

<h3>実装機能</h3>
・店舗リスト　・スタッフリスト　・カレンダー<br>
・予約　・予約確認　・休日設定<br>

カレンダーは当日以降で予約ができ、
予約があった際、Djangoの確認画面で確認をすることが
できます

<h2>◆使用言語◆</h2>
HTML・CSS(Bootstrap)<br>
python(Django)
